## Getting Started

1. **Clone the repo:**
   ```bash
   git clone https://github.com/barundas97/Baruns_Portfolio.git
   cd Baruns_Portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

5. **Preview the production build:**
   ```bash
   npm run preview
   ```
